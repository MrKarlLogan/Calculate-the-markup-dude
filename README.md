# Проект "Calculate the markup, dude"

**НАХОДИТСЯ В РАЗРАБОТКЕ**

### Описание

Внутренний инструмент для команды. Позволяет рассчитывать маржинальную стоимость товаров с учётом акций.
Руководитель управляет позициями (CRUD), отправляет общие уведомления и согласовывает сделки.
Сотрудники просматривают актуальные позиции, рассчитывают наценку и отправляют запросы на согласование.

### Стек технологий

**Frontend:**

- React + TypeScript
- Redux (управление состоянием)
- React Router (навигация)
- SCSS (стилизация)
- WebSocket — уведомления в реальном времени

**Backend:**

- Node.js + Express + TypeScript
- TypeORM (схемы без декораторов) + PostgreSQL
- JWT (аутентификация)
- WebSocket — для согласований и оповещений

**Инфраструктура:**

- Vite (сборка фронтенда)
- Docker (контейнеризация)
